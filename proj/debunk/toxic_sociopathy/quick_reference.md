# Quick Reference Guide - Citation Evaluation

Use this as a cheat sheet while evaluating citations.

---

## Accessibility Quick Codes

| Code | Meaning | Action |
|------|---------|--------|
| **ACC** | Accessible | Proceed with evaluation |
| **PAY** | Paywalled | Try preview/abstract; note if you have access |
| **404** | Dead link | Check Internet Archive |
| **REM** | Removed | Check Internet Archive |
| **RED** | Redirects | Note where it goes |
| **NUL** | No URL | Library/purchase needed |
| **AUTH** | Requires auth | Try to access; note if you have account |

---

## Credibility Quick Assessment

**Ask yourself:**
1. Who published this? (organization/outlet)
2. What are their editorial standards?
3. Do they issue corrections?
4. What's their track record?
5. Any conflicts of interest?

**Rating shortcuts:**
- Government stats, peer-reviewed journals, primary docs = **5**
- NYT, WaPo, Reuters, AP, established news = **4**
- Partisan but fact-based news, reputable think tanks = **3**
- Highly partisan sources, poor fact-checking = **2**
- Conspiracy sites, known misinfo sources = **1**

---

## Representation Quick Check

**Ask yourself:**
1. Does the source actually say this?
2. Is there context I'm missing?
3. Are there caveats/limitations ignored?
4. Is this outdated?
5. Is there contradictory info in the same source?

**Common red flags:**
- ⚠️ Book quote has no quote marks in source
- ⚠️ Source says "may" or "suggests", book says "proves"
- ⚠️ Source is from 2008, presented as current
- ⚠️ Book cites paragraph 1, paragraph 2 contradicts it
- ⚠️ Source studying X population, book generalizes to all

---

## Common Misrepresentation Types

| Type | Example |
|------|---------|
| **Cherry-pick** | Cites one stat, ignores contradictory stats in same source |
| **Out of context** | Quote is real but surrounding text changes meaning |
| **Limitations ignored** | Study says "limited evidence", book says "proves" |
| **Causation confused** | Study finds correlation, book claims causation |
| **Sampling ignored** | Study of college students, book applies to everyone |
| **Author intent reversed** | Source arguing AGAINST claim, book uses it FOR claim |

## Triage Decision Tree

```
Can you access it?
├─ NO → Mark INACCESSIBLE, move to next
└─ YES
   └─ Obvious red flag? (retracted, extremist, satire)
      ├─ YES → Mark red flag, document, move to next
      └─ NO
         └─ Does it say what book claims?
            ├─ CLEARLY YES → Quick credibility check, mark ACCURATE
            ├─ CLEARLY NO → Document misrepresentation, move to next
            └─ UNCLEAR → Add to "deep dive" list
```

## Time Budgets (Suggested)

### Phase 1: Triage (all 182 citations)
- **5 min per citation** = ~15 hours total
- Goal: Accessibility + obvious red flags
- Output: Prioritized list for Phase 2

### Phase 2: Quick Evaluation (120 accessible citations)
- **15 min per citation** = ~30 hours total
- Goal: Credibility + basic representation check
- Skip deep reading, just skim
- Output: Ratings for all accessible sources

### Phase 3: Deep Dive (30 key citations)
- **45-60 min per citation** = ~25-30 hours total
- Goal: Detailed analysis of book's main claims
- Full read, cross-reference, research authors
- Output: Detailed reports

**Total time estimate: 70-75 hours for complete analysis**

---

## Priority Criteria

**Evaluate these first:**
1. Citations the book returns to repeatedly
2. Citations supporting major claims (in titles/chapter headers)
3. Academic/scientific sources (higher credibility if accurate)
4. Easily accessible sources (quick wins)

**Save for later:**
1. Social media posts (often less critical)
2. Paywalled sources (time-consuming)
3. Redundant citations (multiple sources saying same thing)

---

## Pattern Analysis Checklist

After evaluating 30-50 citations, check for patterns:

**Accessibility patterns:**
- [ ] What % are actually accessible?
- [ ] What % are paywalled academic sources?
- [ ] Any clusters of dead links?

**Source type patterns:**
- [ ] What % are news vs academic vs advocacy?
- [ ] Over-reliance on any particular outlet?
- [ ] Missing any types of sources? (e.g., no primary research?)

**Credibility patterns:**
- [ ] Average credibility score?
- [ ] What % are credibility 1-2 (low)?
- [ ] Any patterns in low-credibility sources?

**Representation patterns:**
- [ ] What % are accurately represented?
- [ ] What % are cherry-picked?
- [ ] What % are misrepresented?
- [ ] Common types of misrepresentation?

**Bias patterns:**
- [ ] What % from left vs right vs center sources?
- [ ] Any ideological bubble? (only cites one perspective)
- [ ] What % from advocacy orgs vs neutral sources?

## Red Flags Checklist

**Automatic disqualification:**
- ✅ Source has been retracted
- ✅ Source is from SPLC-designated hate group
- ✅ Source is satire (The Onion, Babylon Bee) presented as fact
- ✅ Source doesn't exist / appears fabricated
- ✅ Quote is misattributed to wrong person

**Major concerns (not automatic disqualification):**
- ⚠️ Study failed to replicate
- ⚠️ Source has issued major correction
- ⚠️ Source predates scientific consensus change
- ⚠️ Author has financial conflict of interest
- ⚠️ Journal is predatory/pay-to-publish

**Minor concerns:**
- ⚪ Source is behind paywall (limits verification)
- ⚪ Source is opinion piece not labeled as such
- ⚪ Source has known political bias
- ⚪ Sample size is small

---

## Common Pitfalls to Avoid

❌ **Don't**: Reject source just because it's biased
✅ **Do**: Evaluate if the facts are accurate despite bias

❌ **Don't**: Accept source just because it's credible
✅ **Do**: Check if book accurately represents it

❌ **Don't**: Assume old = wrong
✅ **Do**: Check if newer evidence contradicts it

❌ **Don't**: Count citations
✅ **Do**: Evaluate quality of citations

❌ **Don't**: Skip sources you can't access
✅ **Do**: Mark them and note you couldn't verify

## Sample Workflow (30 min per citation)

1. **Access** (5 min)
   - Download/open source
   - If can't access: mark and move on

2. **Skim** (5 min)
   - What type of source?
   - Who wrote it?
   - Quick credibility assessment

3. **Find claim** (10 min)
   - Search for keywords from book's claim
   - Read surrounding context
   - Check for caveats/limitations

4. **Compare** (5 min)
   - Does source say what book claims?
   - What's missing from book's representation?
   - Any contradictory info?

5. **Document** (5 min)
   - Fill out YAML
   - Copy key quotes
   - Note follow-up needed

---

## Stopping Points

**Good places to take breaks:**
- After completing a section (all Lie #1 citations)
- After triage phase (before deep evaluation)
- After 3-5 detailed evaluations (intensive work)
- After pattern analysis (natural milestone)

**When to stop for the day:**
- If you're skimming instead of reading
- If you're getting angry (bias creeping in)
- If you're rushing evaluations
- After 3-4 hours of intensive work

---

## Quality Control

Every 10 citations, ask yourself:
- [ ] Am I being consistent in my ratings?
- [ ] Am I documenting enough for others to verify?
- [ ] Am I being fair to sources I disagree with?
- [ ] Am I giving benefit of doubt to ambiguous cases?
- [ ] Would my evaluation hold up to scrutiny?

---

## Final Checklist

Before calling a citation "done":
- [ ] Accessibility status recorded
- [ ] Source type identified
- [ ] Credibility rating with reasoning
- [ ] Representation accuracy assessed
- [ ] Key quotes copied
- [ ] Red flags noted (if any)
- [ ] Follow-up items listed (if any)
- [ ] Time spent recorded
- [ ] Overall support rating assigned
